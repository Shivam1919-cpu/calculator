# Simple Calculator

This is a basic calculator made using HTML, CSS, and JavaScript.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear button
- Delete button

## Files

- index.html
- style.css
- script.js

- ## How to Run

Open `index.html` in any web browser.
